# Sound-Manager Template

This is a template repository for creating a Sound-Manager project. It provides a basic structure and essential files to help you get started quickly with managing sounds.

## How to use

1. Clone the repository:
```bash 
git clone https://github.com/dev-lukas0/Sound-Manager-Template.git
```

2. Navigate to the project directory:
```bash
cd Sound-Manager-Template
```

3. Install dependencies:
```bash
npm install
```

4. Compile the TypeScript code:
```bash
npm run watch
```

5. Start the local Rojo server:
```bash
rojo serve
```
